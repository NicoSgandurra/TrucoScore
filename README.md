# TrucoScore

TrucoScore es una aplicación web sencilla y práctica para llevar los puntos de tus partidas de Truco.

## Características Principales

- **100% Gratuita y Open Source:** El código cuenta con licencia abierta y su uso no tiene ningún costo.
- **Sin Publicidad ni Tracking:** Diseñada pensando en la privacidad y comodidad del usuario. No contiene anuncios molestos ni rastreos ocultos.
- **Acceso Directo Web:** No necesitas descargar nada para empezar a usarla. Simplemente ingresa a la página web y ya puedes llevar los puntos.

## Cómo Usar

Puedes acceder y ejecutar la aplicación directamente desde el navegador entrando al siguiente enlace:

🔗 **[nicosgandurra.github.io/TrucoScore/](https://nicosgandurra.github.io/TrucoScore/)**

## Uso Offline e Instalación en Móviles

TrucoScore está preparada para acompañarte a todos lados, incluso si te quedas sin datos o conexión a internet:

📱 **Instalación:** Si accedes desde tu teléfono móvil, puedes elegir la opción de "Instalar" / "Añadir a la pantalla de inicio". 
🚫 **Modo Offline:** Una vez instalada, la aplicación funcionará de manera completamente **offline**, sin requerir conexión a internet.
💾 **Guardado Local:** Los resultados y el progreso de los puntos se guardarán de forma segura en la memoria de tu dispositivo.
